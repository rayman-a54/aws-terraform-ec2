# AWS EC2 Manual + Terraform Project

## Objective
Launch EC2 manually and using Terraform.

## Manual EC2 Steps
- Logged into AWS
- Opened EC2 service
- Launched instance
- Selected Amazon Linux
- Selected t3.micro
- Allowed SSH and HTTP
- Instance running

## Terraform EC2 Steps
- Installed Terraform
- Installed AWS CLI
- Configured aws configure
- Created main.tf
- Ran terraform init
- Ran terraform plan
- Ran terraform apply
